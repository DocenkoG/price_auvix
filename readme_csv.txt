КРАТКОЕ ОПИСАНИЕ CSV ФАЙЛА

О Файле
~~~~~~~
CSV файл предназначен для импорта информации по товарам в учетные и торговые системы дилеров компании Аувикс.
Файл имеет фиксированный набор полей, описывающих цены, название и артикул товара.
Аувикс может изменить их количество в будущем в сторону увеличения и сохранив, при этом, порядок для уже существующих в настоящий момент.
Первая строка файла содержит названия полей. Вторая и последующая строки содержат данные по товарам.

Описание полей
~~~~~~~~~~~~~~
Поля CSV файла в порядке их следования:

Идентификатор - уникальный код товара по учетной системе Аувикс, артикул. 
Розничная цена в валюте товара - розничная цена товара. Является рекомендованной Аувикс ценой для конечного, розничного покупателя.
Дилерская цена в валюте товара - дилерская цена товара, по прайс-листу Аувикс в валюте товара.
Наличие на складе 0 - нет/1 - есть) - признак наличие товара на складах Аувикс. 1 = товар есть на складах, 0 = товара на складах нет.
Категория - категория товара, по учетной системе Аувикс. Аувикс в любой момент может поменять привязку товаров в категориям или изменить их название. Однако, из практики, изменение категорий происходит редко.
Модель - модель товара.
Название - название товара. В поле, кроме собственно наименования, также содержится информация и об основных потребительских свойствах товара.
Производитель - название бренда.
Валюта товара - учетная валюта товара. Может принимать следующие значения: Рубль, USD, Евро. В случае изменения товарного портфеля или списка поставщиков, список валют может быть расширен. Аувикс не гарантирует постоянство наименований валют.
Дилерская цена в рублях - дилерская цена товара в рублях по курсу Аувикс. Поле эквивалентно "Дилерская цена в валюте товара", но используется валюта рубли, а не валюта учета товара.
